# Solana GIF Portal (Taylor's Version)

**Connect your Phantom wallet then upload and upvote your favourite (not strictly) Taylor Swift GIFs!**

To run app:
```
cd app
export NODE_OPTIONS=--openssl-legacy-provider
npm run start
```
